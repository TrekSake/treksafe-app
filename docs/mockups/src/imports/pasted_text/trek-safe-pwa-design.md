PROMPT

Design a complete Progressive Web Application (PWA) called TrekSafe, focused on hiker safety, expedition tracking, and emergency response management.

The goal is to create a product that feels like a real startup-ready application, not a university project or a form-based administrative system.

Product Vision

TrekSafe helps hikers register expeditions, define an expected return time, and confirm their safe return.

If the hiker does not check in before the expected return time (plus a configurable tolerance period), the system automatically triggers an emergency protocol by notifying emergency contacts and rescue teams.
